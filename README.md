# New User Account Creation Tool 

For local active directory instances.

I originally built this using NodeJS (visible as reid-andy/aduc-user-management). The tool worked, but was built to be lightweight and not need persistent data storage. I'm rebuilding in C# to learn Blazor and deploy on IIS. 

This version will use proper encryption for secrets, a database for storing preferences and job descriptions, and a guided install for sysadmins.
